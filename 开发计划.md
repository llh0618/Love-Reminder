# 女友纪念日助手 — Android 开发计划

## 平台与方案

- **平台**：Android 原生 Kotlin
- **构建**：Gradle Wrapper 命令行（方案二）

- **目标版本**：API 36 (Android 14)
- **存储**：Room (SQLite) 本地数据库
- **提醒**：系统日历 ContentProvider 写入

## 功能模块

### 1. 纪念日管理（首页）
- 倒计时卡片列表，显示距今天数
- 支持类型：情人节 / 520 / 七夕 / 生日 / 纪念日 / 自定义
- 支持重复：每年 / 每年农历 / 仅此一次
- 创建/编辑纪念日时，**自动写入系统日历**（带每年重复规则）
- 提醒完全交给系统日历处理，用户解锁手机就能看到

### 2. 喜好记录 & 讨厌记录（分拆入口）
- **两个独立入口**：底部导航分别是「喜欢」和「讨厌」
- 两个页面共用同一个 Note Entity，通过 category 字段区分 like/dislike
- 记录内容 + 标签（美食偏好/饮品口味/品牌喜好/颜色/爱好等）
- 支持搜索和标签筛选

### 3. Markdown 导出
- 在喜欢/讨厌页面顶部放置「导出 MD」按钮
- 一键将**喜欢 + 讨厌**所有记录合并导出为一个 .md 文件
- 导出格式按分类/标签分组，方便喂给 AI（"帮我根据她喜欢的餐厅类型选个约会地点"）
- 通过 Android ShareSheet 分享到微信/文件管理器

导出格式示例：
```markdown
# 她的喜好手册

## ❤️ 喜欢
### 美食
- 火锅，尤其是重庆老火锅
- 提拉米苏

### 饮品
- 桂花拿铁、少糖奶茶

### 品牌
- ZARA、COS

## 💔 讨厌
### 美食
- 香菜、折耳根

### 其他
- 迟到、打电话不接
```

## 数据模型 (Room Entity)

```kotlin
@Entity
data class Anniversary(
    @PrimaryKey val id: String = UUID.randomUUID().toString(),
    val name: String,
    val date: String,           // "2023-05-20"
    val type: String,           // birthday/valentine/qixi/520/anniversary/custom
    val repeat: String,         // yearly/lunar_yearly/once
    val remindBeforeDays: Int = 0,  // 提前几天提醒（0=当天）
    val note: String = "",
    val calendarEventId: Long = 0  // 写入系统日历后回填的 eventId
)

@Entity
data class Note(
    @PrimaryKey val id: String = UUID.randomUUID().toString(),
    val content: String,
    val category: String,       // like/dislike
    val tags: String = "",      // 逗号分隔
    val createdAt: Long = System.currentTimeMillis()
)

@Entity
data class Gift(
    @PrimaryKey val id: String = UUID.randomUUID().toString(),
    val name: String,
    val link: String = "",
    val status: String,         // want/bought/given
    val note: String = "",
    val createdAt: Long = System.currentTimeMillis()
)
```

## 提醒流程

```
用户新建纪念日
  → Room 保存数据
  → ContentResolver.insert(CalendarContract.Events) 写入系统日历
    - 设置 RRULE (每年重复) 或单次
    - 设置提前提醒 ALARM
  → 回填 calendarEventId
用户编辑纪念日
  → 更新 Room
  → ContentResolver.update 更新日历事件
用户删除纪念日
  → 删除 Room + ContentResolver.delete 删除日历事件
```
